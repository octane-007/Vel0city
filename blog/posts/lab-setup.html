00 — Setting Up Your Pentesting Lab

Posted: June 22, 2025

Before hacking the world, you need a safe playground.

🧰 Tools Required

  • VirtualBox (or VMware)
  • Kali Linux ISO
  • Metasploitable2 / OWASP Juice Shop

🚀 Steps

  1. Install VirtualBox
  2. Download Kali and Metasploitable
  3. Create VMs inside VirtualBox
  4. Connect them on a host-only network
  5. Hack without fear 🔐
